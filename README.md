# Training ROI Analytics

## Overview
This project demonstrates a simple and practical approach to measuring training effectiveness
using pre- and post-training performance scores.

## Problem Statement
Many L&D programs struggle to quantify training impact beyond attendance and completion metrics.
Business leaders require clear evidence of performance improvement to justify training investments.

## Solution
This project calculates average performance improvement by comparing pre-training and post-training
assessment scores, providing a simple proxy for training ROI.

## Metrics Covered
- Pre-training performance score
- Post-training performance score
- Average improvement score

## Tools & Technologies
- Python
- Pandas
- CSV-based sample data

## Sample Use Case
- Sales capability programs
- Leadership development initiatives
- Technical or functional upskilling programs

## Note
All data used is dummy data created purely for demonstration purposes.
No organizational or confidential data is included.
